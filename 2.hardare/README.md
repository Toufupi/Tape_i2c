# hardware

硬件上是比较简单的，计划使用光敏电阻和比较器实现一个简单的检测电路，为了确保稳定，会增加一个补光灯，设计一个3d打印的黑室，确保每次的信号是类似的。

1. 电路部分的设计已经完成，并通过验证，PCB项目位于嘉立创
2. 3D打印的纸袋暗盒也设计完毕并且验证，尺寸需要做一点小修改
